# AI-Weather-Hallucination-Sim
25气象灾害学16组作业，气象预测幻觉模拟器部分Beta版，线上没额度，本地没算例，停用生成气象图功能
本模拟器为智能体应用，由于相关规定，如需使用请联系气象灾害学课程16组成员或使用自己有的大模型密钥

定位：模拟进行一次AI大模型灾情预测，即模拟一次“AI研判-人在回路（Human-in-the-loop）验证” 的协同防灾决策流程。
调用实时监测数据（雷达、自动站、视频）进行交叉验证，或派遣力量进行快速实地核查。将AI的数据处理优势与人类的常识判断、责任意识有机结合，才能构建可信、可靠、可用的智能防灾新范式。

游戏说明：
这是一个基于大语言模型（DeepSeek/Gemini）构建的AIGC式即时文本模拟游戏。
玩家将扮演2025年的一个气象工程师，在AI辅助系统的帮助下预测极端天气并做出判断。
**核心挑战**：AI系统可能会产生“幻觉”（Hallucination），你需要利用气象学知识辨别真伪，做出正确决策。
游戏进度基于恐慌值和回合值，你需要在右侧终端即时发布正确的灾情通报和辟谣信息，才能降低恐慌值，如果不管，恐慌值会增加，做出错误决策也会增加。回合通过左侧选项决策推进，需要判断出是否有AI幻觉出现，并选择正确的选项。（由于幻觉模拟需求，大幅度调高了AI幻觉出现率，所以可能会有一些“生硬”的幻觉，每回合的模拟不是一定会有幻觉，选择两个选项里较好的一个）

游戏文件位于dist文件夹内，游玩前需要输入自己的大模型api key，目前版本仅支持deepseek，gemini（不用输入），气象图部分停用。
